# 配置
- typescript
    - 配置 tsconfig.json
```shell
yarn add -D typescript
```
- 安装 koa 相关
```shell
yarn add -S koa koa-router
yarn add -D @types/koa @types/koa-router @types/node ts-node
```
- 使用 mongoose 连接 mongodb
```shell
yarn add mongoose
yarn add -D @types/mongoose
```
